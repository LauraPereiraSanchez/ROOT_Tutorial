# Tutorial_ROOT

This repository contains different example files to master ROOT:

i) Edit a tree: Read, Clone/Copy with/out cuts, reduce/add branches.
ii) Histograms

To get the necessary ntuples for the tutorials do:
mkdir Files; cd Files; cp /afs/cern.ch/user/l/lapereir/public/TutorialNuples/* .
or change the path "Files" in the tutorials to /afs/cern.ch/user/l/lapereir/public/TutorialNuples/
